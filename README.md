## Overview
This project demonstrates an end-to-end operational reporting solution built using Microsoft Excel. The objective is to convert raw ERP-style sales exports into a clean analytical model and interactive dashboard that supports business decision-making for multi-store retail operations.

## Business Problem
Operations teams often rely on manual Excel reports that are time-consuming, error-prone, and difficult to scale. This project automates data preparation, modeling, and KPI reporting to improve reporting accuracy and turnaround time.

## Tools & Technologies
- Microsoft Excel
- Power Query (ETL)
- Power Pivot (DAX)
- Advanced Excel formulas
- Dashboard design best practices

## Data Pipeline
1. Raw ERP exports ingested using Power Query  
2. Data cleaned, standardized, and enriched  
3. Star-schema style data model created  
4. DAX measures developed for KPIs  
5. Interactive dashboard built for analysis  

## Key Metrics
- Total Sales
- Gross Margin
- Margin %
- Store-level performance
- Time-based trends (MoM, YoY)

## Outcomes
- Reduced manual reporting effort
- Improved data consistency and reliability
- Enabled self-service analysis for business users
- Scalable template for other retail categories
